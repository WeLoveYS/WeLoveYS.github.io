发表于2022-03-23|更新于2022-03-23
|阅读量:

第一节，Unreal的版本安装

    {% note red 'fas fa-fan' modern%}1. 官网下载Epic{% endnote %}
    {% note red 'fas fa-fan' modern%}2. 进入虚幻商城，下载需要的版本{% endnote %}

第二节，下载导入模型所需的插件

3dmax2018

------进入虚幻4官网，在搜索框搜索Datasmith，点击获取插件，安装至3dmax之中

maya2018

------在虚幻商城直接搜索Maya LiveLink，安装至引擎之中

第三节，认识Ue4虚幻引擎

快捷键

------使用F11可全屏游戏运行的界面，此为第一人称射击模版

场景移动

------按住鼠标右键，通过wasd移动，鼠标调控视角，q键下调视角，e键上调视角
------创建项目时勾选新手用户包 StarterContent，运行项目时，需要点开关卡。
