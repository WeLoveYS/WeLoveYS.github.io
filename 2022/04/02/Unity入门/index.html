构建一个时钟

知识总结

  1. 物体的定义
    • [SerializeField] //序列化私有变量
    • Transform hoursPivot, minutesPivot, secondsPivot;
  2. 控制台输出
    • Debug.Log(DateTime.Now.Hour);
  3. 物体的旋转：
    • hoursPivot.localRotation = Quaternion.Euler(0f, 0f, hoursToDegrees * time.Hour);
  4. 获取更加详细的时间：
    • TimeSpan time = DateTime.Now.TimeOfDay;
    • hoursPivot.localRotation =
      Quaternion.Euler(0f, 1f, hoursToDegrees * (float)time.TotalHours);

构建动画图形


知识总结

将序列化整数字段变成滑块

//添加一个可序列整数字段,Range()表示为增加滑块，限定resolution的大小
[SerializeField, Range(10, 100)]
int resolution = 50;//分辨率

物体的缩放

//设置分辨率大小
float step = 2f / 50;
//将缩小后的点赋予给变量
var scale = Vector3.one * step;
point.localScale = scale;

将生成的预制体变成子类

//设置该点为所绑定的Graph的子对象
point.SetParent(transform, false);

让物体成正弦波摆动

//获取时间
float time = Time.time;
//更改点的位置，并让他动起来
position.y = Mathf.Sin(Mathf.PI * (position.x + time));
